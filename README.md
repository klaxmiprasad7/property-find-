# property-find-

🏠 Smart Location-Based Property Search System (India)

A modern, responsive, and feature-rich real estate web application that allows users to search, compare, and list properties across all states in India.

🚀 Features
🔍 Search properties by location, budget, and type
📊 Smart ranking system based on price, amenities & distance
🏘️ Explore properties across all Indian states & cities
⚖️ Compare multiple properties side-by-side
❤️ Save favorite properties
📤 Upload & list your own properties (Buy/Sell/Land)
🔐 Simulated Login & Register system
📱 Fully responsive (Mobile, Tablet, Desktop)
🎨 Modern UI with animations and smooth transitions
🧠 Tech Stack

Frontend

React.js (Vanilla / CDN-based setup)
Tailwind-like custom CSS styling
Framer-style animations (custom CSS animations)

Backend

Node.js (Pure HTTP server, no frameworks)
JWT-based authentication
In-memory database (users + properties)
